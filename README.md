## 📖 项目简介

**短线情绪择时助手**是一个为面试 AI 金融产品经理岗位打造的综合能力展示项目，集成了以下核心能力：

- ✅ **AI/大模型应用**：集成阿里云千问进行智能分析
- ✅ **金融数据分析**：真实市场数据 + 技术指标 + 择时建议
- ✅ **产品设计**：完整的用户流程和可视化展示
- ✅ **实时监控**：盘中实时监控 + 异常预警
- ✅ **持仓管理**：持仓跟踪 + 盈亏监控 + 止损止盈

### 核心功能（12项）

1. **市场情绪评分**：市场宽度 + 资金流向 + 波动率
2. **重大事件识别**：半自动事件分析 + 板块映射
3. **双模式运行**：事件驱动 + 轮动捕捉
4. **选股推荐**：综合评分 + 风险分类
5. **持仓配置**：根据资金量和风险偏好生成配置
6. **止盈止损**：3套方案 + 历史胜率
7. **实时监控**：5分钟更新 + 盘前预警
8. **异常熔断**：极端市场检测 + 风险控制
9. **持仓管理**：持仓跟踪 + 盈亏监控
10. **预警中心**：分级预警 + 历史记录
11. **Web界面**：6大功能页面 + 自动刷新
12. **数据持久化**：JSON存储 + 自动备份

## 🔧 技术架构

### 数据层

- **CMES API**：实时行情数据（延迟约38ms）
- **AkShare API**：备用数据源
- **数据缓存**：5分钟TTL，减少API调用

### 分析层

- **情绪分析**：三维度综合评分
  - 市场宽度（35%）- 涨跌家数比、涨跌停比
  - 资金流向（40%）- 北向资金、主力资金
  - 波动率（25%）- 历史波动率、百分位

- **事件分析**：事件识别 + 板块映射
- **选股算法**：多因子评分 + 风险分类

### 监控层

- **实时监控**：5分钟定时更新
- **预警系统**：三级预警（CRITICAL/WARNING/INFO）
- **熔断机制**：极端市场检测

### 展示层

- **Web界面**：Streamlit + 6个功能页面
- **实时更新**：可配置自动刷新
- **数据持久化**：JSON存储 + 自动备份

## 📊 核心指标说明

### 市场情绪评分

**公式**：`得分 = 市场宽度×35% + 资金流向×40% + 波动率×25%`

**状态分类**：
- ≥80：极度贪婪
- ≥65：贪婪
- ≥45：中性
- ≥30：恐慌
- <30：极度恐慌

### 选股评分

**公式**：`得分 = 情绪×30% + 资金×30% + 市值×20% + 活跃度×20%`

**股票分类**：
- 稳定型：大市值 + 成交量稳定
- 敏感型：大市值 + 成交量波动大
- 活跃型：小市值 + 成交活跃
