### 启用 AI 功能的步骤：

#### 1. 安装 dashscope

```bash
pip install dashscope
```

#### 2. 配置 API Key

```bash
# 在 D:\ai_timing_assistant 目录创建 .env 文件
# 添加以下内容：
DASHSCOPE_API_KEY=your_api_key_here
```

#### 3. 获取 API Key

- 访问：https://dashscope.console.aliyun.com/
- 注册阿里云账号
- 开通千问服务（有免费额度）
- 复制 API Key

#### 4. 重新启动应用

```bash
cd D:\ai_timing_assistant
streamlit run web/app.py
```

#### 5. 在界面中启用 AI

- 勾选侧边栏的"启用 AI 分析"
- 即可看到 AI 生成的深度分析
