## 📋 功能模块详解

### 1. 市场状态分析（自动加载）

**显示内容：**
- 📈 市场趋势：牛市/熊市/震荡
- 📊 趋势强度：百分比显示
- 🎯 判断置信度：分析可靠性
- 💼 仓位建议：具体操作建议

**技术指标：**
- 上证指数得分（0-100）
- 深证成指得分（0-100）
- 创业板指得分（0-100）

**AI 分析（可选）：**
- 智能市场解读
- 趋势原因分析
- 风险提示

---

### 2. 个股择时分析

**输入：** 6位股票代码（如 600519）

**输出：**
- ✅ **操作建议**：买入/持有/卖出/观望
- 📊 **置信度**：0-100%
- 💰 **目标价位**：预期上涨目标
- 🛑 **止损价位**：风险控制线

**评分系统：**
- 技术面得分（MACD、RSI、KDJ、均线）
- 相对强弱（与大盘对比）
- 市场环境影响

**分析理由：**
- 4条核心原因
- AI 智能解释（可选）
- 风险因素提示

---

### 3. 批量分析功能

**输入：** 多只股票代码（每行一个）

**输出：**
- 表格展示所有股票分析结果
- 支持导出 CSV
- 支持导出 HTML 报告

## 📱 使用流程

### 第一次使用：

1. **启动应用**
   ```bash
   cd D:\ai_timing_assistant
   streamlit run web/app.py
   ```

2. **打开浏览器**
   - 自动打开 http://localhost:8501
   - 或手动访问

3. **查看市场状态**
   - 自动分析三大指数
   - 显示当前市场趋势
   - 给出仓位建议

4. **分析个股**
   - 输入股票代码
   - 点击"开始分析"
   - 查看择时建议

5. **批量分析**（可选）
   - 输入多只股票代码
   - 批量获取分析结果
   - 导出分析报告
